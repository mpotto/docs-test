Getting started with Sphinx¶

I will be using Sphinx and Read The Docs to automatize documentation creation and to host the documentation pages. The convention hereafter adopted will be that of numpy. To convert between numpydoc and the rst syntax I will be using the napoleon package.

class random_code.NFWDisk(r_s=24.42, rho_s=0.184)[source]¶

Navarro-Frenk-White (NFW) dark matter denstiy profile. See 1 for more information.

Parameters

  • r_s (float) – Scale radius (kpc).

  • rho_s (float) – Scale density (GeV/cm³).

\[\rho_{\mathrm{NFW}}(r) = \dfrac{\rho_s}{\dfrac{r}{r_s} \left(1+\dfrac{r}{r_s} \right)^2}\]
